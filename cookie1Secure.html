Secure Cookie Example

Using HttpOnly Cookies:Cookies that are used to store sensitive
information like session IDs should be marked with the HttpOnly flag.
This ensures that they cannot be accessed via client-side JavaScript
and are only sent with HTTP requests.

Using SameSite Attribute:The SameSite attribute prevents the browser
from sending this cookie along with cross-site requests, which helps prevent
Cross-Site Request Forgery (CSRF) attacks.
